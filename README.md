# Sliding Character Select for Ikemen GO

Current version is made to work with nightly builds from May 2026.

Adds the ability to add more character slots than the default Ikemen GO engine allows, pressing down on the lowest row or up on the highest row will change the visible slots on the screen.  
  
After the 1st player selects their character it will then allow the 2nd player to select a character and so on until all players have selected a character.

# Installation

Place the sliding character select.lua into the Ikemen external/mods folder.  
  
Add additional characters to the select.def as you normally would, after all slots are filled, more slots will be available when pressing down on the lowest row.
